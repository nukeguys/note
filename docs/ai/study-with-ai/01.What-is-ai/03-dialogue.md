# 03. Dialogue Log

## 대화 1. Symbolic AI는 정말 AI인가?

### 🧠 사용자의 실제 질문 (원문)

> 심볼릭 ai는 ai라기보다는 그저 프로그램에 가까운 것 아닌가?

### 💬 대화 맥락

- AI 정의가 시대별로 달랐다는 흐름을 이해한 직후
- “AI라는 이름이 붙는 기준”에 대한 의문 제기

### 🔎 핵심 논의 요약

- 현대 기준에서는 충분히 타당한 지적
- 하지만 당시에는 “사고를 기계로 구현하려는 최초의 시도”
- AI의 정의는 성능 기준이 아니라 **접근 방식 기준**으로 이동해왔음

### 🔄 사고의 변화

- “AI냐 아니냐” → “당시 무엇을 목표로 했는가”
- AI를 정적인 개념이 아니라 **역사적 과정**으로 보기 시작

## 대화 2. Adaptation과 학습의 의미

### 🧠 사용자의 실제 질문 (원문)

> ‘특징 추출’이란 무엇을 중요한 정보로 볼지 결정하는 것 이고,  
> ML에서 사람이 지정한다는건 사람이 어떤 특징이 중요하다는 걸 미리 지정해 두었다는 의미인가요?

### 💬 대화 맥락

- Symbolic AI와 ML의 차이를 이해하던 중 등장
- “학습”의 실제 의미를 정확히 짚으려는 질문

### 🔎 핵심 논의 요약

- 전통 ML에서는 사람이 “무엇이 중요할지”를 미리 정의
- 딥러닝은 이 결정 자체를 데이터로부터 학습
- Adaptation은 단순 반응이 아니라 **경험에 따른 행동 변화**

### 🔄 사고의 변화

- 학습을 “지식 추가”가 아니라
- **행동 전략의 갱신**으로 이해하게 됨

## 대화 3. 자율성은 기술적으로 가능한가?

### 🧠 사용자의 실제 질문 (원문)

> 자율성은 기술적으로 구현 가능할까. 상상이 되지 않는데.  
> 그리고 만약 기계가 자율적인 것 처럼 보인다면 그건 또 어떻게 봐야할까?

### 💬 대화 맥락

- 지능의 기준이 지식 → 행동 → 자율성으로 이동
- 인간과 AI의 경계를 어디에 둘 것인가에 대한 고민

### 🔎 핵심 논의 요약

- 기술적으로 “자율적으로 보이게” 만드는 것은 가능
- 하지만 내부적으로는 여전히 규칙 + 학습 결과
- 인간의 행동 역시 학습된 결과일 수 있다는 관점 등장

### 🧩 남겨둔 질문

- 인간의 자유의지는 환상인가?
- 자율성의 기준은 내부 메커니즘인가, 외부 관찰인가?
