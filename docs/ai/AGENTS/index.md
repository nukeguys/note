# AGENTS.md

[AI Agent 협업을 위한 문서 설계 방법론](../index.md)을 기반으로 실제 `AGENTS.md`를 생성하기 위한 프롬프트이며, 다음 과정을 통해 생성되었습니다:

1. AI agent Best Practice, SDD(Spec-Driven Development)등 관련 자료들을 NotebookLM의 리소스로 추가한 후 방법론의 초안을 작성
2. 초안을 Claude, Gemini, ChatGPT에게 각각 전달해 피드백을 수집한 후 Claude가 정리
3. 그렇게 작성된 [AI Agent 협업을 위한 문서 설계 방법론](../index.md)을 ChatGPT에게 요약해서 프롬프트 초안 생성
4. 작성된 초안을 Claude에게 전달하고 보완하여 v1.0 프롬프트 작성
5. 작성된 내용에 대해 Gemini와 ChatGPT의 피드백을 수집하고 Claude가 반영하는 과정을 반복하여 개선된 버전을 작성

> 리소스가 추가되거나 변경되는 경우 [AI Agent 협업을 위한 문서 설계 방법론](../index.md)도 변경될 수 있으며, 이후 동일한 과정을 거쳐 새로운 버전(v2.0)을 생성할 예정
