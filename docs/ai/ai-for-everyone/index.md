## 정리 방법

### 주차별 강의노트와 자막(en/ko)를 모두 notebooklm에 리소스로 추가하고 1차 데이터 추출

#### 강의에서 나온 모든 핵심 내용의 목록

```text
이 강의에서 실제로 설명된 모든 핵심 내용을 순서대로 빠짐없이 정리해줘.

조건:

- 요약하지 말 것
- 개념, 규칙, 주장, 이유 설명, 예시는 모두 포함
- 강의 흐름을 따라가며 정리
- 슬라이드에 있는 항목 중
- 강의에서 설명된 것만 포함
```

#### 슬라이드를 기준으로 섹션 구조 명확화

```text
슬라이드의 각 항목이
강의 자막에서 어떻게 설명되었는지 매칭해줘.

- 자세히 설명된 항목
- 간단히 언급만 된 항목
- 슬라이드에는 있지만 설명되지 않은 항목
으로 구분해줘.
```

#### 용어 정리

```text
강의에서 사용된 주요 기술 용어를 모두 추출하고,
영문 표현과 한글 자막 표현을 함께 정리해줘.
```

### Claude에 NotebookLM 결과와 원본 자막 추가 후 정리

```text
다음은 약 10분 분량의 영어 강의 자료를 정리하기 위한 입력이다.

입력 자료:
1. NotebookLM에서 추출한 강의 핵심 내용 목록
2. NotebookLM에서 정리한 슬라이드 ↔ 강의 매칭 결과
3. NotebookLM에서 정리한 핵심 용어 사전
4. 영어 강의 자막(txt)
5. 한글 강의 자막(txt)

요청:
요약하지 말고, 강의 노트(교재)처럼 내용을 정리해줘.

정리 기준:
- 기준 텍스트는 영어 자막(txt)
- 한글 자막은 의미 확인 및 이해 보조용
- NotebookLM 결과는 누락 방지 및 구조 참고용
- 슬라이드 구조를 섹션 구분 기준으로 사용
- 강의에서 설명된 핵심 내용은 모두 포함
- 불필요한 반복, 말버릇, 중복 표현만 제거
- 강의의 설명 흐름(순서)을 유지

표현 규칙:
- 중요한 용어는 영어 + 한글 병기
- 각 개념은 아래 구조를 따를 것:
  · 개념 이름
  · 정의 (강의에서 설명한 의미)
  · 왜 중요한지 / 배경
  · 강사가 든 예시 또는 비유
- 슬라이드에만 있고 설명되지 않은 내용은
  [슬라이드 언급만 있음]으로 표시

출력 형식:
1. 강의 제목 / 주제
2. 강의 목적
3. 전체 구조(슬라이드 기준 섹션)
4. 섹션별 강의 노트
5. 개념 간 연결 관계
6. 강사가 강조한 핵심 포인트

```
