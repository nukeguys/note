# 지능(Intelligence)이란 무엇인가?

## 1. 개념적 정의

- **일반적 의미**: 문제 해결, 학습, 적응, 추론 능력
- **인지과학적 관점**: 환경과 상호작용하며 목표를 달성하는 능력
- **AI 관점**: 입력 데이터를 바탕으로 예측/결정/생산을 수행하는 시스템적 능력

## 2. 인간 지능 vs 기계 지능

| 특징      | 인간 지능                   | 기계 지능 (AI)                |
| --------- | --------------------------- | ----------------------------- |
| 학습 방식 | 경험, 관찰, 추론            | 데이터 기반 통계적 학습       |
| 적응력    | 유연, 직관적                | 특정 영역 내 규칙/패턴 학습   |
| 이해      | 의미 이해, 암묵 지식 활용   | 확률적 패턴 인식, 의미는 없음 |
| 창의성    | 기존 지식의 조합, 발상 전환 | 학습 데이터 범위 내 생성      |

## 3. 지능의 구성 요소 (AI 관점)

1. **인지(Cognition)**: 입력 데이터 해석 및 의미 추론
2. **학습(Learning)**: 경험을 통한 모델/패턴 개선
3. **추론(Reasoning)**: 목표 달성을 위한 선택과 판단
4. **적응(Adaptation)**: 환경 변화에 따른 전략 조정

## 4. 지능 정의의 철학적 쟁점

- **Strong AI vs Weak AI**
  - Strong AI: 기계가 인간처럼 의식과 이해를 가짐
  - Weak AI: 특정 작업 수행 능력만 존재
- **지능과 의미(Understanding)**
  - Searle의 Chinese Room 논증: 계산만으로 진정한 이해 가능할까?
    - [Searle, John. _Minds, Brains, and Programs_ (Chinese Room 논증)](https://zoo.cs.yale.edu/classes/cs471/13s/13s-papers/searle-chinese-room-bbs.pdf)
    - [인공지능은 의식을 가질 수 있을까? | 중국어 방 사고실험 - 지식은 날리지](https://www.youtube.com/watch?v=F6donYNLTMw) (간단히 이해하기 좋다.)
- **지능의 본질**
  - 문제 해결 능력, 패턴 인식, 목표 달성 능력 중 무엇이 핵심인가?

## 5. AI 연구와 지능

- 초기 AI: Symbolic AI (규칙 기반)
  - 목표: 인간처럼 사고하는 기계
  - 한계: 현실 세계의 불확실성과 복잡성
- 현대 AI: Machine Learning / Deep Learning
  - 목표: 데이터 기반 예측과 패턴 인식
  - 특징: 의미는 모르지만 통계적/확률적 판단 가능
- LLM 등장: 대규모 언어 데이터로 ‘언어적 추론’ 수행
  - 문제: Hallucination, 의미 이해 여부 논란

## 6. 핵심 정리

- 지능 = **문제를 해결하고 목표를 달성하는 능력**
- 인간 지능은 의미와 직관을 포함
- AI 지능은 데이터 기반 통계적 추론에 가까움
- “AI가 인간처럼 지능적일 수 있는가”는 여전히 논쟁적
